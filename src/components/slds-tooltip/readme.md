# Tooltip ![avaliability](https://img.shields.io/badge/avaliability-available-green.svg)  ![testing](https://img.shields.io/badge/coverage-0%25-red.svg) ![documentation](https://img.shields.io/badge/documentation-draft-red.svg)

A Tooltip is a small piece of contextual information about an element on the screen, which is displayed when a user hovers or focuses on the element it is describing. It is not focusable and cannot contain focusable content.

## Props

| Name      | Type    | Required | Default | Description |
| -------- | -------- | -------- | ------- | ----------- |
| propName | PropType | false    |         | Prop description. |

## Slots

| Name    | Optional | Description |
| ------- | -------- | ----------- |
| default | true     | Slot description |


## Events

| Name       | Arguments    | Description                            |
| ---------- | ------------ | -------------------------------------- |
| event-name | argumentName | Fired when foo. |
